# Web Technologies Final Project Portfolio

A 6-page responsive portfolio website built for the Certificate IV in IT
(Programming) Web Technologies final project. The site is themed around
Davisual Studio, my freelance videography business, and showcases both
video work and front-end development skills.

## Pages
- Home: hero, featured projects, client feedback, and links to all pages
- Projects: video portfolio grid
- About: background and toolkit
- Resume: education, skills and experience
- Contact: form with vanilla JS regex validation
- Sample Project: API Ninjas form (Dictionary and Dad Jokes) using fetch

## Built with
- HTML5 (semantic markup)
- Tailwind CSS v3
- Vanilla JavaScript
- Font Awesome (icons)
- Git and GitHub Pages
